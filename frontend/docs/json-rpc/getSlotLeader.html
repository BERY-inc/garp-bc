getSlotLeader

Return the leader identity for the given slot, or current slot if omitted.

Request

{
  "jsonrpc": "2.0",
  "method": "getSlotLeader",
  "params": { "slot": 12345 },
  "id": 1
}

Response

{
  "jsonrpc": "2.0",
  "result": "garp:validator-xyz",
  "id": 1
}

Error

{
  "jsonrpc": "2.0",
  "error": { "code": -32602, "message": "Invalid params" },
  "id": 1
}

Notes

  • If slot is omitted, the node returns the current slot leader.
