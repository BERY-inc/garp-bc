simulateTransaction

Dry-run a transaction command and return execution logs and effects.

Request

{
  "jsonrpc": "2.0",
  "method": "simulateTransaction",
  "params": {
    "command": {
      "type": "transfer",
      "from": "garp:alice",
      "to": "garp:bob",
      "lamports": 1000
    }
  },
  "id": 1
}

Response

{
  "jsonrpc": "2.0",
  "result": {
    "logs": [
      "invoke transfer",
      "debit alice: 1000",
      "credit bob: 1000"
    ],
    "accounts": {
      "garp:alice": { "pre": 10000, "post": 9000 },
      "garp:bob": { "pre": 500, "post": 1500 }
    },
    "ok": true
  },
  "id": 1
}

Error

{
  "jsonrpc": "2.0",
  "error": { "code": -32603, "message": "Simulation failed" },
  "id": 1
}

Notes

  • Simulation does not commit changes; it estimates execution effects and logs.
  • Exact fields may expand as schemas are standardized; consult SDK types when available.
