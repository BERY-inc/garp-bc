sendTransaction

Submit a transaction command to the participant node for processing.

Request

{
  "jsonrpc": "2.0",
  "method": "sendTransaction",
  "params": {
    "command": { "type": "transfer", "from": "garp:alice", "to": "garp:bob", "lamports": 1000 }
  },
  "id": 1
}

Response

{
  "jsonrpc": "2.0",
  "result": {
    "signature": "5Yk...",
    "status": "Submitted"
  },
  "id": 1
}

Error

{
  "jsonrpc": "2.0",
  "error": { "code": -32000, "message": "Transaction rejected" },
  "id": 1
}

Notes

  • Ensure accounts and signatures are valid; malformed commands may be rejected.
  • Use SDK helpers to build commands where possible.
