getBalance

Return the balance for a given participant or account id.

Request

{
  "jsonrpc": "2.0",
  "method": "getBalance",
  "params": { "participantId": "garp:alice" },
  "id": 1
}

Response

{
  "jsonrpc": "2.0",
  "result": 1500,
  "id": 1
}

Error

{
  "jsonrpc": "2.0",
  "error": { "code": -32602, "message": "Invalid params" },
  "id": 1
}

Notes

  • Balance may be returned as number or string depending on SDK encoding.
