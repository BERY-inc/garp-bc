getBlock

Fetch a block by slot or hash.

Request

{
  "jsonrpc": "2.0",
  "method": "getBlock",
  "params": { "slot": 12345 },
  "id": 1
}

Response

{
  "jsonrpc": "2.0",
  "result": {
    "slot": 12345,
    "hash": "9xJ...abc",
    "parentHash": "7kP...def",
    "timestamp": 1731000000,
    "transactions": [
      {
        "signature": "5Yk...",
        "status": "Confirmed",
        "accounts": ["garp:alice", "garp:bob"],
        "commands": [{ "type": "transfer", "lamports": 1000 }]
      }
    ],
    "logs": ["program log: ..."],
    "stateCommitment": "Cmmt...xyz"
  },
  "id": 1
}

Error

{
  "jsonrpc": "2.0",
  "error": { "code": -32602, "message": "Invalid params" },
  "id": 1
}

Notes

  • Provide either slot or hash; if both provided, slot takes precedence.
  • Field names may evolve as schemas are standardized; use SDK models where possible.
